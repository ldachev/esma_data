ESMA Securities Liquidity Dashboard

A Python and Streamlit project for exploring ESMA-style FIRDS instrument reference data, FITRS liquidity and transparency fields, and European trading venue MIC metadata.

GitHub Pages hosts this static project overview. The live interactive dashboard should be run with Streamlit locally or deployed to a Streamlit-compatible hosting service.

Run locally Deploy on Streamlit Community Cloud

Search and Filter

Filter by ISIN, instrument name, asset class, MIC, venue type, country, liquidity, and date.

Local Analytics

Processed data is stored in DuckDB, with SQLite fallback for lightweight environments.

ESMA Ready

Uses public ESMA register indexes and keeps a fallback sample so streamlit run app.py works.

Run Locally

Install dependencies with pip install -r requirements.txt, then start the dashboard with streamlit run app.py.
